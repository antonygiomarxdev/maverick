## Decisions Made

- FOpts extraction from PHY payload: FCtrl lower nibble contains FOptsLen (0-15 bytes)
- MAC command CID 0x02 = LinkCheckReq/LinkCheckAns per LoRaWAN 1.0.x spec Section 5
- RX1 at +1000ms, RX2 at +2000ms from uplink receive time per LoRaWAN Class A specification
- DownlinkScheduler designed as generic over DownlinkRepository and RadioTransport traits (hexagonal)
